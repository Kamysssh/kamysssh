## Что я делаю

- **Корпоративные и клиентские ассистенты (RAG)** — ответы опираются на ваши документы и регламенты, а не на «фантазии модели».
- **Ускорение и экономика** — кэширование повторяющихся запросов, чтобы снизить задержки и стоимость обращений к LLM.
- **Каналы внедрения** — Telegram-боты, API-сервисы (в т.ч. на **FastAPI**), сценарии интеграции с сайтом и внутренними системами.
- **Качество ответов** — оценка RAG-пайплайна метриками (**RAGAS** и связанные практики), чтобы понимать, где система ошибается и что улучшать.
- **Быстрые пилоты без «тяжёлого» бэкенда** — связки **n8n** + векторное хранилище (**Pinecone**) + LLM для проверки гипотез.
- **Мультимодальные сценарии** — текст, голос, изображения: удобнее для пользователя и меньше трения на входе в сервис.
